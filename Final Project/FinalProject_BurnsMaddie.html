About This Project

Welcome to the Fayetteville Theatre Recommender, a web application designed as my final project for CTI110. This site helps users discover the best theater in Fayetteville based on their preferences for size, type of performance, and location. It utilizes an interactive quiz to generate theater recommendations, and it is powered by HTML, CSS, and JavaScript.

This website is designed to help theater lovers in Fayetteville find a theater that matches their needs, whether they prefer large musicals, intimate plays, or Shakespearean performances. Additionally, it provides contact information and links to local theaters' websites to help users find more details and schedule a visit.
